## truffle 开发

### 安装

官方文档：
https://trufflesuite.com/docs/truffle/quickstart.html

```s
npm i -g truffle
mkdir MetaCoin
cd MetaCoin
truffle unbox metacoin
```

测试
```s
truffle test --debug
```
